# Gameplay Player Model

These GLB files were generated from the baseball character created by the user
in Meshy on July 19, 2026. They are original project assets and do not contain
models, textures, skeletons, or animations extracted from a commercial baseball
game.

- `player_base.glb`: skinned player, recolorable uniform materials, `Pitch_R`
- `player_run.glb`: animation-only `Run` clip
- `player_walk.glb`: animation-only `Walk` clip

Rebuild them with `tools/BaseballCharacterPipeline/build_character_asset.py`.
